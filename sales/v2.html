Proof shape

Credibility comes from acceptance criteria, not broad promises.

Architecture review Assessment and prioritization
Output Architecture summary, risk register, decision matrix, and backlog-ready task breakdown.
Acceptance signal A designated engineer can explain prioritized next steps without undocumented verbal clarification.
Terraform foundation Infrastructure baseline
Output State strategy, module structure, environment layout, command flow, rollback notes, and handover guide.
Acceptance signal Non-production validation can be executed from the documented handover guide.
CI/CD bootstrap Safer release flow
Output Trigger matrix, approval guideline, pipeline blueprint, failure path, and rollback guidance.
Acceptance signal Non-production validation and production gate simulation have explicit documented steps.
Cost hygiene Baseline visibility
Output Cost driver breakdown, label coverage snapshot, candidate list, risk notes, and recurring review checklist.
Acceptance signal Safe optimization proposals can be prepared with stop conditions and rollback guidance.
